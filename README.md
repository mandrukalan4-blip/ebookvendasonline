# Do Zero às Vendas Online

Landing page do produto digital "Do Zero às Vendas Online".

## Descrição
Esta página foi criada para apresentar o eBook, destacar seus benefícios e direcionar o usuário para a compra.

## Objetivo
Ajudar iniciantes a conhecer o produto e acessar a oferta de forma simples.

## Estrutura
- Página de vendas
- Botão de compra
- Seções de benefícios
- Apresentação do conteúdo

## Publicação
Projeto hospedado na Vercel.
